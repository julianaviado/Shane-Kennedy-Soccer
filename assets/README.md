# Assets

Images and other media for the site go here — for example:

- `hero.jpg` — Shane coaching a player (wide/landscape, used at the top of the page)
- `shane.jpg` — headshot or coaching portrait (About section)
- `action.jpg` — keeper making a save (full-width band)
- Logo files (SVG or PNG)
- Web font files (WOFF2), if self-hosting fonts instead of Google Fonts

Reference files from the page like: `<img src="assets/hero.jpg" alt="...">`
